CSS 伪类选择器（Pseudo-class Selectors）

CSS 伪类选择器（Pseudo-class Selectors）用于选择元素的 特定状态 或 结构位置，而不是直接基于 HTML 标签、类名或 ID 进行选择。它们以 :（单冒号）开头，用于增强交互性和样式控制。

1. 常见伪类选择器

(1) 用户交互状态

伪类 作用 示例
:hover 鼠标悬停时 a:hover { color: red; }
:active 鼠标点击时 button:active { background: blue; }
:focus 获取焦点时 input:focus { border-color: green; }
:visited 已访问的链接 a:visited { color: purple; }

示例：按钮悬停效果

(2) 结构位置选择

伪类 作用 示例
:first-child 父元素的第一个子元素 li:first-child { font-weight: bold; }
:last-child 父元素的最后一个子元素 li:last-child { color: gray; }
:nth-child(n) 第 n 个子元素 li:nth-child(2) { color: red; }
:nth-of-type(n) 同类型的第 n 个元素 p:nth-of-type(2) { font-size: 1.2em; }

示例：表格隔行变色

(3) 表单相关

伪类 作用 示例
:checked 选中的复选框/单选框 input:checked + label { color: green; }
:disabled 禁用的表单元素 input:disabled { opacity: 0.5; }
:valid 输入合法的表单 input:valid { border-color: green; }
:invalid 输入不合法的表单 input:invalid { border-color: red; }

示例：选中的复选框样式

2. 伪类 vs 伪元素

特性 伪类（Pseudo-class） 伪元素（Pseudo-element）
语法 :（单冒号） ::（双冒号，CSS3 推荐）
作用 选择元素的 状态 选择元素的 特定部分
示例 :hover, :nth-child() ::before, ::after, ::first-line

3. 高级用法

(1) :not() 反向选择

(2) :is() 或 :where() 批量选择

(3) :has()（实验性，部分浏览器支持）

4. 总结

  • 伪类选择器 用于选择元素的 状态（如 :hover）或 结构位置（如 :nth-child）。
  • 常用伪类：
    • 交互状态：:hover, :active, :focus
    • 结构位置：:first-child, :nth-child(n)
    • 表单相关：:checked, :disabled
  • 伪元素（::before, ::after）用于选择元素的特定部分。

掌握伪类选择器可以让你的 CSS 更灵活，实现更丰富的交互效果！ 🚀
